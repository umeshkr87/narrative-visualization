# Meat Consumption Trend of the World

A narrative visualization built with D3.js exploring global meat consumption trends and their relationship to economic growth.

**Live demo:** https://umeshkr87.github.io/narrative-visualization/

## About

This project is a multi-scene narrative visualization that answers three questions:

1. **Where** in the world is meat consumption highest?
2. **How** has per-capita meat consumption changed over time?
3. **Why** — specifically, how does a country's economic growth relate to its meat consumption?

The visualization uses a **martini-glass structure**: guided scenes walk the viewer through the analytical argument, ending in a free-exploration view where users can drill into individual countries.

## Built With

- **D3.js** for data binding, scales, and transitions
- **HTML / CSS / JavaScript** (vanilla, no framework)
- Scene-based navigation with annotations to guide interpretation
